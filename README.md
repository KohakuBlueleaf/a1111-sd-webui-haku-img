# a1111-sd-webui-haku-img
An Image utils extension for A1111's sd-webui

Features:
* blending (support 17 mode)
* color adjust
* tone curve
* blur
* img2sketch(thourgh XDoG)
* pixelize(Kmeans or dithering)
* glow (blur and blend)
* inoutpaint support tool
